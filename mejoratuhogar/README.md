# MejoraTuHogar.es
Proyecto web para una empresa local en Castellón enfocada en:
- Compra y venta de propiedades
- Alquiler de viviendas
- Venta de ventanas PVC (producto diferencial)

## Objetivo principal
Generar leads cualificados mediante:
- Formularios
- WhatsApp
- Llamadas

## Estrategia clave
Posicionamiento como:
"Inmobiliaria moderna + mejora del hogar"

## Stack principal
- Astro (frontend)
- Sanity (CMS)
- DonDominio (dominio + DNS + email)
